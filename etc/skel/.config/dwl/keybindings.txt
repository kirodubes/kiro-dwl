── 2. Window Management ───────────────────────────────────────────────────────
  super + Q               Close window
  super + shift + Q       Close window
  super + J               Focus next in stack
  super + K               Focus previous in stack
  super + I               More master windows
  super + shift + I       Fewer master windows
  super + H               Shrink master area
  super + L               Grow master area
  super + ctrl + Return   Zoom to master
  super + F               Toggle fullscreen
  super + shift + Space   Toggle floating
  super + left-click      Move window
  super + right-click     Resize window
  super + middle-click    Toggle floating

── 3. Workspaces / Tags / Groups ──────────────────────────────────────────────
  super + 1..9            View tag 1..9  (AZERTY: same physical keys)
  super + shift + 1..9    Move window to tag 1..9
  super + ctrl + 1..9     Toggle tag 1..9 in view
  super + 0               View all tags
  super + shift + 0       Put window on all tags
  super + Tab             Last tag
  super + comma           Focus left monitor
  super + period          Focus right monitor
  super + shift + comma   Send window to left monitor
  super + shift + period  Send window to right monitor
